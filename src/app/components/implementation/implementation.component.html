Para incorporar o widget de feedback em seu site, primeiro adicione esta tag de script ao HTML:

<script defer src = "https://vokzfeedback.app/main.js?id={{ userData.id }}" > </script>

Para adicionar o botão padrão ao seu site, adicione o seguinte código HTML:

<button data-vokz>
  Feedback
</button>

Se quiser identificar qual usuário enviou feedback, você pode opcionalmente adicionar o atributo data-user ao botão e passe o email ou id do usuário atualmente autenticado.

<button data-vokz data-user="idOuEmail">
  Feedback
</button>

É isso, agora você está com o Vokz Feedback configurado.
